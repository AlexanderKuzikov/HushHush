## 📌 Суть проекта

**HushHush** — десктопное приложение для Windows 10/11.  
Цель: кастомный слайдер для демонстрации фото/картинок из выбранного каталога.  
Замена стандартному Photos slideshow — с исправлением его багов и расширенным контролем.

---

## 🛠 Стек

| Слой | Технология | Причина выбора |
|---|---|---|
| Backend / логика | Go | Быстрое чтение ФС, горутины для предзагрузки, маленький бинарник |
| Frontend / UI | HTML + CSS + JS | Знакомый стек, полный контроль над внешним видом |
| Desktop bridge | Wails v2 | Нативный WebView2 (встроен в Win10/11), сборка в один .exe без Electron |
| Настройки | JSON (`%APPDATA%`) | Простое хранение конфига |

**Отвергнутые варианты:**
- ❌ Electron — тяжёлый (~150MB), сложная сборка
- ❌ NW.js — те же проблемы что Electron
- ❌ PyQt6 — не принципиально отвергнут, но Go предпочтительнее
