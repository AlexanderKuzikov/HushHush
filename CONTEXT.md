## 📌 Суть проекта

**HushHush** — десктопное приложение для Windows 10/11.  
Цель: кастомный слайдер для демонстрации фото/картинок из выбранного каталога.  
Замена стандартному Photos slideshow — с исправлением его багов и расширенным контролем.

---

## 🛠 Стек

| Слой | Технология | Причина выбора |
|---|---|---|
| Backend / логика | Go | Быстрое чтение ФС, горутины для предзагрузки, маленький бинарник |
| Frontend / UI | HTML + CSS + JS | Знакомый стек, полный контроль над внешним видом |
| Desktop bridge | Wails v2 | Нативный WebView2 (встроен в Win10/11), сборка в один .exe без Electron |
| Настройки | JSON (`%APPDATA%`) | Простое хранение конфига |

**Отвергнутые варианты:**
- ❌ Electron — тяжёлый (~150MB), сложная сборка
- ❌ NW.js — те же проблемы что Electron
- ❌ PyQt6 — не принципиально отвергнут, но Go предпочтительнее

## ✅ Согласованный функционал

### Обязательно (MVP)
- [ ] Выбор папки с изображениями
- [ ] Полноэкранный показ изображений
- [ ] Настраиваемый интервал смены кадра
  - Слайдер / поле ввода прямо в UI
  - Горячие клавиши `+` / `-` во время показа
  - Диапазон: 1 сек — 60+ сек
- [ ] Предзагрузка следующих 2–3 кадров в фоновом горутине
- [ ] Зацикливание слайд-шоу
- [ ] Режим Shuffle (случайный порядок)
- [ ] Запоминание последней папки и настроек
- [ ] Чистый экран — UI-элементы скрыты в режиме показа
- [ ] Элементы управления появляются только по клику или `Esc`

### Желательно (v2)
- [ ] Эффекты переходов (fade, slide)
- [ ] Поддержка нескольких папок
- [ ] Исключение отдельных файлов из показа
- [ ] Показ имени файла / даты (опционально)
- [ ] Пауза при движении мыши
- [ ] Drag & Drop папки на окно
- [ ] Разные интервалы для разных папок

### Поддерживаемые форматы
JPG, JPEG, PNG, GIF, WebP, AVIF

## 🐛 Баги Photos (которые мы исправляем)

| # | Баг | Наше решение |
|---|---|---|
| 1 | Зависание при переходе к следующему изображению «без причины» | Предзагрузка 2–3 следующих кадров в Go-горутине |
| 2 | Всплывающие элементы управления и полоски поверх изображения | UI показывается только по явному действию пользователя |

## 📋 Соглашения

- Язык кода: Go (backend) + JS (frontend)
- Язык комментариев и документации: русский
- Конфиг хранится в `%APPDATA%/HushHush/config.json`
- Ветка разработки: `main` (до появления фич-веток)
- Коммиты: `тип: описание` (например `feat: предзагрузка изображений`)
