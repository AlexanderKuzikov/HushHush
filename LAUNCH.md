# Первый запуск HushHush

## Требования

| Инструмент | Версия | Ссылка |
|---|---|---|
| Go | 1.26.5+ | https://go.dev/dl/ |
| Node.js | v24 LTS | https://nodejs.org |
| Wails CLI | v2.12.0 | `go install github.com/wailsapp/wails/v2/cmd/wails@latest` |

## Порядок первого запуск

```bash
# 1. Клонировать репо

git clone https://github.com/AlexanderKuzikov/HushHush
cd HushHush

# 2. Загрузить Go-зависимости (создаст go.sum)
go mod tidy

# 3. Загрузить npm-зависимости
cd frontend && npm install && cd ..

# 4. Проверить окружение (WebView2, Go, Node.js)
wails doctor

# 5. Запустить в dev-режиме
wails dev
```

## Что проверить при первом запуске

- [ ] Окно открылось без ошибок
- [ ] Появляется экран с иконкой папки и кнопкой «Открыть папку»
- [ ] После выбора папки изображения отображаются
- [ ] Смена кадров по таймеру работает
- [ ] Клик по экрану показывает/скрывает панель управления
- [ ] Клавиши `←` `→` переключают кадры
- [ ] `+`/`-` меняют интервал
- [ ] `F` — fullscreen
- [ ] `S` — shuffle
- [ ] `Esc` — скрывает панель
- [ ] Drag & Drop папки на окно

## Сборка .exe

```bash
wails build
# Результат: build/bin/HushHush.exe
```
